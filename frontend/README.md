# Conflux ERC777 Token Transfer DAPP

## Project setup

``` bash
npm install
```

and Chrome extension `Conflux Portal` is required.

### Compiles and hot-reloads for development

``` bash
npm run serve
```

### Compiles and minifies for production

``` bash
npm run build
```
